# Ethan Young

Hey！！，我是 **Ethan Young**， 现在在南京。

在生活中，我扮演着多个角色——程序员、家人、朋友，以及一个对生活充满好奇的普通人。

经历了10年的职场生活，一直想用文字记录下这些角色中的点滴感悟，却总被琐碎的日常牵绊。

这次借重建博客的契机，正式开启写作之旅，记录生活、分享技术、表达思想、见证成长。

本次重建使用了<a className='link link-info' href="https://astro.build/">Astro</a>,部署在
 <a className='link link-info' href="https://vercel.com/arferiks-projects">Vercel</a>
如同程序员的风格，追求简洁与效率。

就像那些伟大的故事，总要从点滴开始。我愿从今天起，用文字拾起人生的碎片，与那些在生活中摸爬滚打的人们共鸣。写作或许不能改变什么，但它是我与世界对话的方式，也是我对自己的一种交代。

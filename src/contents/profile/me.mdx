
# Ethan Young

[english version](/en)

👋 大家好，我是 Ethan Young，一名常驻南京的程序员，也是一位在生活中持续迭代的普通人

十年的职场生涯中，我扮演着多重角色——写代码的工程师，陪伴家人的儿子与丈夫，倾听朋友的伙伴，以及在终端之外，试图理解生活底层逻辑的思考者。

一直想为这些角色中的片段与感悟打上时间戳，却总被琐碎的日常“卡住线程”。这一次，我决定按下写作的启动键，把它作为一个长期运行的 side project：记录生活、分享技术、表达思考、见证成长。

这个博客部署在 <a className='link link-info' href="https://vercel.com/arferiks-projects">Vercel</a>，就像一行行推送到生产环境的代码，力求简洁、稳定而不失温度。

写作对我而言，是一次又一次的 git commit，也是一场场自我 debug。它不求重构世界，只为在信息洪流中留下一个清晰的注释。

伟大的系统从来不是一夜成型，它们诞生于一次次小版本的更新。我也想从今天起，把生活拆成一个个模块，用文字去记录、反思、重构，与这个世界温柔对话。
